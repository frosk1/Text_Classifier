                    #################################
                            Annotations-Daten
                    #################################


Die Annotation dieses Korpus besteht darin, nach einem gesehenen Textpaar
den qualititav hochwertigeren Text auszuwählen. Für eine effiziente
Annotation steht das "anno" Softwaretool zur Verfügung.
https://github.com/frosky/Anno


Die Ordner "Jan" und "Mel" enthalten die manuelle Annotation der Korpora.
Die automatisch generierten Annotationen der Textpaare, befinden sich in
dem Ordner "auto_sample".

Die Ordner "auto_sample_geteilt" und "manuelle_anno_geteilt" beinhalten
die aufgeteilten Annotationen.



Annotations Schema :

Textpaar    Annotation
...         ...


Beispiel :

Text 267, Text 374		1
Text 397, Text 401		1
Text 189, Text 301		0
Text 226, Text 304		0
Text 275, Text 419		1
